# proyecto_Franz_Zeballos

Protótipo de alta fidelidad de un marketplace de economía circular.

Instrucciones rápidas:

1. Instala dependencias: `npm install`
2. Ejecuta en modo desarrollo: `npm run dev`

El prototipo usa Next.js + Tailwind y componentes básicos para Onboarding, Auth, Home, Publicar, Detalle de producto, Perfil y Comunidad.

Deploy (URL pública)
- Temporal (túnel durante el desarrollo): https://eager-ears-laugh.loca.lt
- Despliegue en Vercel: https://proyfranz.vercel.app

Notas:
- El despliegue a Vercel está configurado mediante GitHub Actions. Añade los secretos `VERCEL_TOKEN`, `VERCEL_ORG_ID` y `VERCEL_PROJECT_ID` en Settings → Secrets para activar deploys automáticos.
